# TMUX 

## Instruction 
It's a terminal reuse tool 


## Install it and configure it 

`sudo pacman -s tmux`

`sudo apt-get install tmux`


## How to use it 

- `ctrl + \  -`   : split window 
- `ctrl + \  |`   : split window

- `ctrl + \  [h/j/k/l]` : cursor move to another split window

- `ctrl + d `     : exit tmux session 

- `ctrl + \ c`    : create a window 
- `ctrl + \ p`    : switch window 
- `ctrl + \ n`    : ~~~~~~~~~~~~~~

- `ctrl + \ ,`    : rename window 


- `tmux list-sessions`

- `ctrl + \ (`    : switch session 
- `ctrl + \ )`    : switch session 
- `ctrl + \ $`    : rename session
